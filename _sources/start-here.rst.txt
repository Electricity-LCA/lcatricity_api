Start Here
=========

What is this?
-------------
An API to estimate real-time environmental impacts using electricity generation
data reported by national and super-national bodies.

Getting started
---------------
1. Install and setup, by going to :ref:`installation-setup`
2. For the Open API documentation go to _a Online API documentation: <http://example.lcatricity.live:8000/docs/>`
3. For development, check out the detailed documentation of the Python classes under :ref:`lcatricity_api`
